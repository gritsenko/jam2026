# Текущее состояние — что реально работает в коде

> **Рабочий док (зеркало кода).** Описывает поведение прототипа **как оно
> реализовано сейчас**, а не как задумано в концепте. Любое изменение логики боя,
> экономики или правил → обновляй этот файл (см. [../README.md](../README.md),
> раздел «Правило синхронизации»). Источник истины — код в `../../src/`, не этот
> текст; при расхождении правь док.

## Что реализовано

Базовый бой поверх каркаса. Headless-симуляция в
[BattleSim.ts](../../src/game/BattleSim.ts) (без Pixi) — сцена только рендерит:

- **Волны врагов.** Скрипт волн — [waves.ts](../../src/config/waves.ts). Враги
  выходят из-за нижнего края экрана и идут по кольцевой дороге
  ([path.ts](../../src/game/path.ts), путь `ENEMY_PATH` в
  [combatRules.ts](../../src/config/combatRules.ts)).
- **Атака башен.** У каждой атакующей карты свой **радиус в клетках** (`rangeCells`),
  бьёт только ближнюю дорогу, по «лидеру», энергоболтами.
- **Грейд.** Ап грейда у разных карт качает разный стат (`upgrade`:
  `power`/`tempo`/`range` — урон/темп/радиус, см. `towerStats`). Радиус
  показывается эллипсом при перетаскивании карты до установки.
- **HP / смерть / награда.** Награда — только **золото**. Утечка врага →
  урон **Core Integrity**. Победа/поражение с баннером.
- **Энергосеть.** Перегрузка энергии замедляет темп всех башен; Overdrive
  (сжигание карты) временно поднимает ёмкость.
- **Рука.** По умолчанию **3 слота** (расширяемо позже).
- **Экономика.** Только награды за килл/волну. Трат золота/кристаллов нет.

Тюнинги — в [combatRules.ts](../../src/config/combatRules.ts),
[enemies.ts](../../src/config/enemies.ts), [cards.ts](../../src/config/cards.ts).
Стартовая раскладка платформы и ресурсы берутся из мока
[battleState.ts](../../src/config/battleState.ts).

Баланс монстров текущего ростера — [enemy-balance.md](enemy-balance.md).
Статы и стихии карт — [cards.md](cards.md) (синергии/мердж в этом доке пока
**проектные**, в коде не реализованы — см. ниже).

## Чего ещё НЕ в коде

Эти системы спроектированы в `../planned/`, но в прототипе их нет — **не считай
их рабочими**:

- Таблица стихийных **синергий**/реакций (Паровой Выброс и т.п.).
- **Мердж** карт в грейды.
- Эффекты карт-поддержки (барьер Щита, замедление Стабилизатора).
- Карты глобальной модернизации.
- Трата золота/кристаллов, реролл (экономика — только награды).
- **Очков синергии (SP) в игре нет** — установка карт гейтится только свободным
  слотом; реальная «цена» — нагрузка на энергосеть.
- Дерево технологий и мета-прогрессия ([../planned/progression-and-tech-tree.md](../planned/progression-and-tech-tree.md)).
- Продажа башен ([../planned/sell-towers.md](../planned/sell-towers.md)).
- Уровни после туториала ([../planned/levels-post-tutorial.md](../planned/levels-post-tutorial.md)).
